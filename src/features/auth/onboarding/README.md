# The Journey (Onboarding Architecture)

Welcome to the **Onboarding Ascension Flow**. This module handles the critical phase where a verified account transitions into a fully established profile inside the academy ecosystem.

## 🌌 The "Ascension" Aesthetic
The onboarding process discards the sterile, form-based data collection methods of the past. Instead, it is an guided "Ascension." Users are prompted to formally define their path (Sensei vs. Gakusei) inside an immersive, cinematic wizard layout. 

## ⚖️ The Core Experience
- **Kinetic Step Engine:** At the top of the interface, the Step Progress bar provides satisfying, fluid momentum that reacts instantly via physical CSS calculations as the user advances forward.
- **Antigravity Roles:** The role selection interfaces deploy Antigravity physics. As the user hovers over their desired path (Teacher or Student), the cards lift iteratively from the background canvas while casting deep shadow blooms, reacting dynamically to their mouse.
