# LiveClassrooms Module: Project Antigravity

Welcome, Lead Sensei. 

This document outlines the vision, user experience, and high-level functionality of the **LiveClassrooms** module within the Nihongo Hub ecosystem.

## 🌌 The Vision
The LiveClassrooms module is the heart of a "Future-Gen" Japanese learning hub. It is designed to shatter the traditional, flat boundaries of online education by merging **3D immersion** with cutting-edge **glassmorphic design**. 

We are not just building a classroom; we are building an interactive, hyper-modern dojo where students feel physically and mentally engaged in the linguistic environment.

## ✨ The Aesthetic: "Antigravity"
Our design philosophy, "Antigravity," emphasizes:
- **Luminescence:** Deep Indigo and vibrant Rose neon accents that breathe life into the UI.
- **Translucency:** High-quality backdrop blurs (Glassmorphism) that float above the spatial plane.
- **Kinetics:** Floating, cinematic Kanji elements that react and shift, making the application feel alive and responsive.

### 🎭 Interactive Legend
Keep an eye on the background—our cinematic Kanji elements are not static! They dynamically respond to **mouse movement** and **scroll depth**, creating a parallax-driven 3D illusion that anchors the "Antigravity" aesthetic.

## 🚀 Feature Set
The LiveClassrooms ecosystem is packed with futuristic teaching tools:
1. **Real-time Video & Audio:** Latency-free WebRTC infrastructure allowing seamless vocal and visual feedback.
2. **Global Bulletins:** Real-time push notifications and broadcast messaging for the entire cohort.
3. **Student Roster Management:** Dynamic participant tracking, active listener visualization, and host controls.
4. **The Resource Vault:** Secure, instant file sharing and interactive workbook drops.

## ⚡ Quick Start Interface
Going live is designed to be frictionless for the teacher. 

1. **Dashboard Initialization:** Enter the Teacher Dashboard. Your "Active Grid" network will automatically sync and establish secure connections to the global mesh.
2. **Select Target Broadcast:** Choose the pre-configured Batch or Class session block from the generated glassmorphic grid.
3. **Execute & Broadcast:** Click **"Resume Class"** or **"Initialize Live."** The system will launch the secure Room, allocate WebRTC sockets, and open the gateway for your students. Welcome to the grid!
